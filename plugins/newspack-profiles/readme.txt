Turn your Google Sheets or Airtable data into beautiful, SEO-optimized profile pages.

== Description ==

Newspack Profiles is a powerful WordPress plugin that transforms data from Google Sheets or Airtable into dynamic profile pages. Perfect for newsrooms and organizations looking to showcase people, places, or any structured data.

**Key Features:**

* **Multiple Data Sources** - Import data from Google Sheets or Airtable
* **Profile Collections** - Create multiple collections with different data sources
* **SEO Optimized** - Automatically generates SEO-friendly URLs and metadata
* **Customizable Templates** - Use pre-built patterns or create custom layouts
* **XML Sitemaps** - Automatic sitemap generation for better search engine indexing
* **Import Management** - Easy-to-use interface for importing and updating data on disconnection
* **Gutenberg Integration** - Native block editor support for displaying profiles

**Requirements:**

* WordPress 6.7 or higher
* PHP 8.1 or higher
* Remote Data Blocks plugin (automatically required)
* Object Cache (optional for performance)

== Installation ==

1. Upload the `newspack-profiles` folder to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Install and activate the Remote Data Blocks plugin if not already installed
4. Navigate to Newspack Profiles in the admin menu to create your first profile collection

== Frequently Asked Questions ==

= What data sources are supported? =

Currently, Newspack Profiles supports Google Sheets and Airtable as data sources.

= Can I customize the profile page layouts? =

Yes, the plugin includes multiple pre-built patterns and supports custom templates through WordPress block themes.
